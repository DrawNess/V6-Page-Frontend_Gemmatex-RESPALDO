Columnas del producto padre

Columna Obligatoria Descripción
productSlug Siempre Identificador del producto. Ej: camisa-oxford
productName Si es nuevo Nombre del producto
productImageUrl Si es nuevo URL de la imagen del producto base
subcategoryId Si es nuevo ID entero de la subcategoría

Columnas de la variante

Columna Obligatoria Descripción
sku Sí Código único de la variante
brand Sí Marca
description Sí Descripción larga (mín. 10 caracteres)
shortDescription Sí Descripción corta (mín. 10 caracteres)
imageUrl Sí URL de la imagen principal
price Sí Precio en Bs (punto decimal: 185.50)
stock Sí Cantidad disponible
unitOfMeasure Sí Unidad: unidad, kg, litro
dimensions Sí Dimensiones o presentación
tags Sí Etiquetas separadas por coma
outlet Sí true o false
colorId Opcional ID del color (dejar vacío si no aplica)
discountPrice Opcional Precio con descuento
galleryUrls Opcional URLs adicionales separadas por coma
is_active Opcional true por defecto

Nombres alternativos aceptados

El sistema acepta encabezados en español:

sku = codigo brand = marca description = descripcion price = precio stock = inventario, cantidad tags = etiquetas outlet = es_outlet, oferta colorId = color_id, id_color imageUrl = imagen dimensions = dimensiones, medidas unitOfMeasure = unidad subcategoryId = id_subcategoria

Regla del producto padre
  • La primera fila de cada productSlug nuevo debe tener productName, productImageUrl y subcategoryId.
  • Las filas siguientes del mismo slug pueden dejar esas 3 columnas vacías.
  • Si el productSlug ya existe en la BD, esas columnas siempre pueden ir vacías.
Upsert automático (crear o actualizar)
Situación Resultado
productSlug ya existeEl producto no se modifica, solo se usa como padre
sku ya existeLa variante se actualiza
sku no existeLa variante se crea
productSlug no existeEl producto se crea (requiere name, image, subcategoryId)
Siempre valida primero (Dry-Run)

Antes de aplicar cambios reales, usa el modo "Validar (dry-run)" para detectar todos los errores sin tocar la base de datos. Corrige y repite hasta que esté limpio.

}
